# Kids Karaganda

Статический сайт-каталог образовательных центров, спортивных секций и творческих кружков для детей в Караганде.

## Демо-вход

- Родитель: `parent@example.com` / `parent123`
- Образовательный центр: `center@example.com` / `center123`

Карточки, добавленные образовательным центром, сохраняются в `localStorage` текущего браузера и сразу появляются в каталоге.

## Публикация на GitHub Pages

1. Загрузите `index.html`, `styles.css` и `app.js` в корень репозитория.
2. Откройте `Settings` -> `Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Выберите ветку `main` и папку `/root`.
5. Сохраните настройки.

После публикации сайт будет доступен по адресу:

`https://kumashevaaizhan-lgtm.github.io/123/`
